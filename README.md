# turbocluster

A GPU-accelerated Python library for filtering Arepo simulations.

<img src="images/turbocluster_banner.png" width="auto">

**NOTE**: This project is in active development and not officially released publicly yet. Expect kinks and bugs. Refer to [this arXiv paper](https://arxiv.org/pdf/2601.06250) if you would like to know more about its use.
